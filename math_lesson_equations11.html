Тема 01

Логарифмические уравнения

Уравнения, содержащие логарифм неизвестного. Ключевой метод — потенцирование и проверка ОДЗ.

📖 Основные формулы и методы

log_a(f(x)) = log_a(g(x)) → f(x) = g(x), при f(x)>0, g(x)>0, a>0, a≠1 log_a(f(x)) = b → f(x) = a^b Формулы логарифмов: log_a(mn) = log_a(m) + log_a(n) log_a(m/n) = log_a(m) − log_a(n) log_a(m^n) = n·log_a(m) log_a(b) = log_c(b) / log_c(a) [формула перехода]
  • ⚠️ ОДЗ: аргумент логарифма должен быть > 0
  • Метод 1: потенцирование (если основания одинаковы)
  • Метод 2: замена переменной (t = log_a(x))
  • Метод 3: логарифмирование обеих частей

Тема 02

Иррациональные уравнения

Уравнения с переменной под знаком корня. Метод — возведение в степень с обязательной проверкой.

📖 Основные методы решения

√f(x) = g(x) → { f(x) = [g(x)]², g(x) ≥ 0 } √f(x) = √g(x) → f(x) = g(x) при f(x) ≥ 0 ⁿ√f(x) = g(x) → f(x) = [g(x)]ⁿ (если n чётное: g(x) ≥ 0; если n нечётное: любое g(x))
  • ⚠️ ОДЗ: подкоренное выражение ≥ 0 (при чётном корне)
  • ⚠️ После возведения в степень — ВСЕГДА проверка
  • Метод: изолировать корень, возвести в степень, решить, проверить
  • Метод замены: t = √f(x), t ≥ 0
Лёгкий уровень (1–7)

Тема 03

Дробно-рациональные уравнения

Уравнения, содержащие переменную в знаменателе. Главное — исключить значения, обнуляющие знаменатель.

📖 Метод решения

P(x)/Q(x) = R(x)/S(x) Шаг 1: Найти ОДЗ — все x, при которых Q(x) ≠ 0, S(x) ≠ 0 Шаг 2: Умножить обе части на НОК знаменателей Шаг 3: Решить полученное целое уравнение Шаг 4: Проверить, не входят ли корни в исключённые значения
  • ⚠️ ОДЗ: все значения, при которых знаменатели ≠ 0
  • Посторонние корни могут появиться — проверка обязательна
  • Метод: приведение к общему знаменателю

Скорость vs точность

На ОРТ/НЦТ лучше сначала решить лёгкие задачи, заработать баллы, а потом вернуться к сложным. Не застревай на одном уравнении дольше 3 минут.

🎯

Метод подстановки

Если не умеешь решать аналитически — подставь варианты ответа (из теста) в уравнение. Быстрее, чем кажется!

🔍

ОДЗ — всегда первым

Запиши ОДЗ до начала решения. Так ты сразу видишь, какие ответы заведомо лишние, и не тратишь время на их проверку.

📊

Формула перехода логарифмов

log₅(25) = ln(25)/ln(5) — используй, когда основания разные. На калькуляторе это можно проверить мгновенно.

🌱

Иррациональные: 2 условия

√f(x) = g(x) всегда даёт два условия: f(x) = [g(x)]² И g(x) ≥ 0. Если второе условие не выполнено — корень лишний.

🧮

Замена переменной — твой друг

В сложных логарифмических уравнениях введи t = log_a(x). Квадратное уравнение относительно t часто решается проще.

🗂 Все ключевые формулы

log_a(a) = 1
Логарифм числа по своему основанию
log_a(1) = 0
Логарифм единицы всегда равен 0
a^(log_a(x)) = x
Основное тождество логарифма
log_a(b·c) = log_a(b)+log_a(c)
Логарифм произведения
log_a(b^n) = n·log_a(b)
Логарифм степени
√(x²) = |x|, не x!
Корень из квадрата = модуль
1/(x-a) → x ≠ a
ОДЗ дробно-рациональных
ln(x) = log_e(x)

Типичные ошибки (не повторяй!)

  • ❌ Ошибка #1 — Забыли ОДЗ логарифма
    log(x-2) = 0 → x-2 = 1 → x = 3. Верно! Но если ответ x = 2 — он не входит в ОДЗ (аргумент = 0).
    ✅ Всегда проверяй: аргумент должен быть строго больше нуля.
  • ❌ Ошибка #2 — Забыли условие g(x) ≥ 0
    √(x+3) = x-1. Возвели: x+3 = x²-2x+1 → x²-3x-2 = 0 → x = (3±√17)/2. Проверка нужна!
    ✅ Подставь каждый корень в исходное уравнение и убедись, что правая часть ≥ 0.
  • ❌ Ошибка #3 — Делят обе части на x
    x² = 5x → делим на x → x = 5. Потеряли корень x = 0!
    ✅ Не делить на выражение с переменной. Переноси и выноси: x²-5x = 0, x(x-5) = 0.
  • ❌ Ошибка #4 — Посторонние корни в дробных уравнениях
    1/(x-2) = ... → после умножения получили x = 2. Но x = 2 обнуляет знаменатель!
    ✅ Проверяй найденные корни: не должны входить в исключения ОДЗ.
  • ❌ Ошибка #5 — Неверное потенцирование
    log₂(x) = log₂(x+1) → x = x+1 → 0 = 1. Уравнение не имеет решений (правильно!), но ошибка: нельзя "добавлять" разные степени.
    ✅ Приводи логарифмы к одному основанию перед потенцированием.
